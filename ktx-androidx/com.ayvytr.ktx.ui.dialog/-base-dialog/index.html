BaseDialog

abstract class BaseDialog(context: Context, themeResId: Int = R.style.TransparentDialog_DimBg) : Dialog

对话框基类，默认对MIUI等某些定制系统手机的Dialog宽度很窄的问题做了处理：isFullWidth默认为true。如果 需要dialog有左右边距，请直接在布局加margin即可.

注意：指定dialog的gravity和坐标left和top时，可这样使用：

override fun show() {
    super.show()
    window?.apply {
    val lp = attributes
    lp.gravity = Gravity.LEFT or Gravity.TOP
    lp.x = left
    lp.y = top
    window?.attributes = lp
    }
}
